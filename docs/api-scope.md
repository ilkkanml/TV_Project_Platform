# API Scope

Compact API scope reference.

## Allowed API Areas

- Health
- Auth
- Users
- Roles / authorization
- Devices
- Licenses
- Subscriptions
- Payments
- Resellers
- Reseller credit transactions
- Remote config
- App versions
- Audit logs
- Temporary encrypted own-device transfer when enabled

## Forbidden API Areas

- Provider inventory
- Stream proxy
- Stream relay
- Stream hosting
- Transcoding
- CDN stream routing
- Channel catalog
- Channel packages
- Public marketplace
- Shared profile library
- Public profile search
- Provider credential sharing
- Permanent backend profile authority by default
