# API Scope

## Allowed API Areas

- Health
- Auth
- Users
- Devices
- Licenses
- Subscriptions
- Payments
- Resellers
- Credits
- Remote config
- App versions
- Temporary encrypted profile transfer

## Forbidden API Areas

- Stream proxy
- Stream relay
- Playlist provider
- Channel catalog
- Transcoding
- CDN edge routing
- Media scraping
- Provider credential sharing
