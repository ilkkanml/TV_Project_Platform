## 2. Active Product Direction

Nexora is a Core Media Player Ecosystem.

Project structure:

- `TV_Project_Platform`: account, device, license, admin, remote config, app version, and temporary profile transfer center.
- `TV_Project`: first Android TV / Fire TV client.
- Future clients: Android Mobile, iOS, Apple TV, Samsung Tizen, LG webOS, and Web Player.

## 3. Non-Negotiable Boundary

The ecosystem must not become a content provider.

Forbidden directions:

- Content hosting.
- Broadcasting.
- Channel package selling.
- Stream relay or proxying.
- Stream transcoding.
- Provider scraping.
- Credential sharing.
- DRM bypass.
- Unauthorized playlist/source collection.

Allowed model:

- User-owned sources.
- Legally authorized sources.
- Local client-side profile ownership.
- Temporary encrypted profile transfer only.

## 4. Free Launch Rule

The first Android TV / Fire TV app remains free until final release level.

During free launch:

- Payment enforcement is not production blocking.
- Subscription payment gates do not block initial usage.
- Account, device, and license infrastructure can be built.
- License checks may return free-launch access states.
- Payment logic remains additive and disabled by default.
