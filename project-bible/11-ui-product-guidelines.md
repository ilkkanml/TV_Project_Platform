# UI Product Guidelines

## Tone

The UI should communicate licensed software access, device control, subscription status, and account management clearly.

## Avoid

- Channel package language
- Content sales language
- Stream access promises
- Unauthorized content implications
- Aggressive claims about media availability

## Prefer

- License active
- Device activated
- Subscription status
- App version
- Remote configuration
- User-provided playlist profile
- Encrypted device transfer
