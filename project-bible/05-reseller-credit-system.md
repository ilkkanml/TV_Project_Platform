# Reseller Credit System

## Purpose

The reseller credit system allows approved resellers to allocate license value or subscription time to customers.

## Core Concepts

- Reseller account
- Credit balance
- Credit transaction ledger
- Customer assignment
- Subscription extension
- Admin adjustment
- Audit log

## Rules

- Credits are platform license credits, not content credits.
- Credits must not imply access to channels or streams.
- Every credit mutation should be auditable.
- Negative balances should be prevented unless explicitly approved by admin policy.
