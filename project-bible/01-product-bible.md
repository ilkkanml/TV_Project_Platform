# Product Bible

## Product Summary

TV Project Platform is a commercial management platform for a licensed IPTV player application.

It manages accounts, subscriptions, device activation, license verification, reseller operations, payments, app versions, remote configuration, and optional user-owned playlist/profile transfer.

## What The Product Is

- Licensed player management platform
- Customer account platform
- Device activation platform
- Subscription and license platform
- Reseller credit platform
- App operations platform
- Optional temporary encrypted web-to-device profile transfer bridge

## What The Product Is Not

- IPTV provider
- Broadcast provider
- Stream host
- CDN
- Relay
- Transcoder
- Channel seller
- Playlist seller
- Content source

## Primary Users

- Customers using the licensed player app
- Resellers managing customer credits and activations
- Admin operators managing platform operations

## Core Product Areas

- Public website
- Customer dashboard
- Reseller dashboard
- Admin dashboard
- API backend
- Shared validation and types
- Infrastructure placeholders
