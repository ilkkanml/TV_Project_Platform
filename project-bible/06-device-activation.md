# Device Activation

## Purpose

Device activation links a licensed customer account to a player device.

## Core Concepts

- Activation code
- Device identifier
- Device name
- User ownership
- License status
- Activation limit
- Revocation

## Rules

- Device activation verifies software access only.
- Device activation does not validate media rights.
- Device activation does not supply streams.
- Device activation must support revocation and audit history.
